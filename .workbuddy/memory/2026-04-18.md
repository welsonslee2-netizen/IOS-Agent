## 技术调研结果
- Python 3.13 已官方支持 iOS 平台（PEP 730）
- BeeWare/Briefcase 可将 Python 打包为原生 iOS App
- iOS 上运行 Python 的主流方案：Pyto、Pythonista 3、a-Shell、iSH
- LLM 推理方案：云端 API / MLC LLM 本地运行 / Core ML
- 构建 iOS App 必须在 macOS 上进行（需要 Xcode）
- 用户当前环境为 Windows，需要 macOS 环境（云主机/GitHub Actions）

## 推荐技术栈
- 核心框架: BeeWare (Briefcase + Toga UI)
- Python 运行: CPython 3.12+
- LLM: OpenAI/DeepSeek API 或 MLC LLM 本地
- Agent 框架: LangChain / LlamaIndex
- 构建环境: macOS + Xcode 15+
